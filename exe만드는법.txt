<파이썬 exe 실행파일 만드는 법>
개요: 파이썬 코드를 exe 실행 파일로 만들려면
1. PyInstaller 도구 사용.
2. GUI(Tkinter)파일 선택창이 있는 코드가 있어야 함.

<실행파일 생성 전 패키지 확인>
pandas같은 외부 라이브러리 사용한다면 누락되면 안됨
cmd> pip install selectolax requests pandas tqdm openpyxl

<exe 실행파일 만드는 과정>
1. Vscode 열고 
cmd> pip install pyinstaller

2. 터미널에서 exe 실행파일이 위치한 폴더로 이동한 후, 아래 명령 실행
pyinstaller --onefile --windowed content_checker.py

# onefile = 여러파일을 하나의 exe로 묶는 옵션
# windowed 실행 시 검은 창 안 띄움 (GUI프로그램만 띄움)

3. exe 파일 위치
dist 폴더가 자동으로 생성됨.
